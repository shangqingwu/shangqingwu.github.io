用hexo搭建博客

Jan 19, 2017 | 搭建博客 | Hits

hexo+github搭建一个自己的博客

可以参考：搭建博客-百度

安装hexo

  1. 首先安装git、node两个软件。安装完之后。->按windows+r ->输入cmd ->输入 git --version和node -v查看安装版本（有版本说明安装成功）。
  2. 输入npm install hexo -g通过npm在全局安装hexo。
    • npm（管理包）、 全局（-g：-global）

      搭建本地网站

  3. 创建hexo文件夹：
    • 新建文件夹（名称一定为英文 djjblog）->在文件夹里面 shift+右键 ->打开命令窗口 ->输入hexo init
  4. 输入npm install安装依赖包
  5. 输入hexo s -g-> 启动服务器，出现本地博客网址 -> http://localhost:4000
    • 注意： 这个窗口一定不能关闭，因为hexo s -g是用来启动服务器的，如果关闭了，那么服务器也就关闭了，就没法把文章上传到博客上去了。
    • s -> server(服务器) 、 g -> generate(形成)
  6. 再在相同文件夹中再重新打开一个命令窗口。

    搭建远程博客

  7. 登录github官网-> 新建Repository(仓库) -> name为：用户名.github.io->(shangqingwu.github.io) -> 复制生成的地址
  8. 修改配置文件

  • 路径：djjblog/_config.yml

    1. 修改里面的deploy（部署）属性中的type:git、repository(仓库):刚才复制的地址、branch(分支):master属性。

      • 注意： type、repository、branch三个属性必须对齐。
      1. 设置一下博客中的内容
  1. 输入hexo d -g 完成部署 -> 远程、本地博客都推上去
  • 远程博客地址：http://shangqingwu.github.io
  • d -> deploy(部署) 、g -> generate(形成)
  1. 每次修改东西之后，都要hexo d -g 一下。

    更换博客主题

  2. hexo s -g-> 启动服务器
  3. 再重新开一个命令窗口
  4. 进http://hexo.io -> theme -> 选择主题 -> 点开主题图片上的小眼睛，看主题安装的注意事项 -> 点主题 -> 复制下面的install(安装)地址
  5. 在命令窗口中输入复制的install(安装)地址，如果主题需要安装渲染器，也要执行一下渲染器命令
  6. 将webstorm中djjblog/_config.yml 文件中的theme属性改为所选的主题名称
  7. 输入hexo d -g ，把修改的内容推到远程和本地博客上。
    如果本地没更新，则先阻断一下服务器（ctrl+c），再重新启动服务器（hexo s -g）。

    往博客上推笔记

  8. hexo n "笔记名" 创建新笔记
  9. 用webstorm打开文件夹（djjblog）-> 路径：source/_posts/笔记名.md -> 编辑就可以了
    • categories:类别 、tags:标签名 -> 用于分类、标记笔记，便于查询。
  10. 每次编辑完 -> hexo d -g ，把修改的内容推到远程和本地博客上。（hexo g 只是推到本地博客上） -> 刷新页面

    给博客添加新的About(关于)页面

  11. hexo n page "about" 创建新about页面
  12. 用webstorm打开文件夹（djjblog）-> 路径：source/about/index.md -> 编辑就可以了
  13. 每次编辑完 -> hexo d -g ，把修改的内容推到远程和本地博客上。（hexo g 只是推到本地博客上） -> 刷新页面
