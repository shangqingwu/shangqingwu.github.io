null和undefined的区别

Mar 23, 2017 | Hits

null和undefined的区别

  • null和undefined都代表没有，但是null是属性存在但是值不存在，undefined是连这个属性都不存在；
  1. null：空对象指针，没有具体的值，一般都是我们手动设置初始的值为null，后期的话会给其赋值；-> 现在没有，以后可能会有；
  2. undefined：未定义，连东西都没有，一般是浏览器默认的值；-> 自始至终都不会有；
  • document.parentNode -> null 浏览器天生自带的一个属性：父亲节点的属性，返回null，因为一个页面中的document已经是最顶级元素了，他没有父亲；
  • document.parentnode -> undefined document没有parentnode属性，返回undefined；

null

  1. 我们设定一个变量，后期需要使用，那么前期我们设置默认值为null；

    var total=null;
function sum(num1,num2){
    total=num1+num2;
    return total;
}
var res=sum(1,2); 
console.log(res); //3

  2. 在JS内存释放中，我们想释放一个堆内存，就让其值变为null即可；

      var obj={name:"kim",age:18};
obj=null;

  3. 我们通过DOM中提供的属性和方法获取页面中的某一个元素标签，如果当前这个标签不存在，获取的结果是null，而不是undefined；

    document.getElementById("div1");  /*null  -> 如果页面中没有id名为div1的元素，那么就返回null*/

  4. 在用正则的exec方法和字符串的match方法进行捕获时，如果当前要捕获的字符串和正则不匹配（字符串中没有所需要的内容）时，捕获到的结果为null；

      var reg=/\d+/g;
var str="kimSyu";
reg.exec(str); //null
str.match(reg); //null

undefined

  1. 在JS预解释的时候，只声明未定义，默认返回undefined；

      console.log(num); //null
var num=13;

  2. 在一个函数中，如果没有写return，或者return后面没有写东西，默认返回undefined；

     function fn(){
}
var res=fn();
console.log(res);  //undefined

  3. 函数中设置了形参，但是执行的时候没有往里面传实参，默认返回undefined；

     function fn(a,b){
   console.log(a);  //10
   console.log(b);  //undefined   
}
fn(10);

  4. 获取一个对象的属性名对应的属性值，如果当前的这个属性名不存在，属性值默认返回undefined；
    -> 可以应用这个道理来检测当前浏览器是否兼容某一个方法!!!

      var obj={};
console.log(obj.name); //undefined

检测当前浏览器是否兼容getComputedStyle 方法

window.getComputedStyle -> 获取当前元素经过浏览器计算过的样式；

  • 在谷歌中，我们获取的结果是：function getComputedStyle() { [native code] }
  • 在IE6-8中，我们获取的结果是：undefined， -> 因为在这个浏览器中不兼容window.getComputedStyle 方法；
//getCss方法，有些样式属性在不同浏览器中是不兼容的，如：opacity
function getCss(curEle,attr) {
    var val=null,reg=null;//val->私有 reg -> 私有变量
    if("getComputedStyle" in window){//兼容
        val=window.getComputedStyle(curEle,null)[attr];
    }else{//不兼容IE6-8
        if(attr==="opacity"){//如果传递进来的结果是opacity，说明我想获取到的是透明度(opacity: 0.1;)，但它在IE6-8中不兼容，因此需要使用滤镜filter来进行处理（先在css中写 filter:alpha(opacity=10);）；
            val=curEle.currentStyle["filter"];// -> "alpha(opacity=10)" 要把获取到的结果进行剖析，获取里面的数字，让数字除以100才能和标准的浏览器保持了一致；
            reg=/^alpha\(opacity=(\d+(?:\.\d+)?)\)$/i;
            val =reg.test(val)?reg.exec(val)[1]/100:1;
        }else {
            val=curEle.currentStyle[attr];
        }
    }
    reg=/^-?\d+(\.\d+)?(px|pt|rem|em)$/i;
    return reg.test(val)?parseFloat(val):val;
}
var res=getCss(box, "opacity");
console.log(res);  //0.1
