1.3.2 面向对象

面向对象

  1. 对象是一个抽象的概念，类似于我们的自然界。自然界中分为了人类、动物类、植物类…，而我们每一个人都是人类中的一个实例；
  • 对象：js中万物皆对象，它是一个泛指 ;
  • 类：对对象的具体细分;
  • 实例：某一个类别中具体的一个事物;
  1. 基于类和实例来研究js就是面向对象。
  2. 内置类：
  • 数据类型类：Number、String、Boolean、Object、Array、RegExp、Date、Function...
  • HTMLCollection(元素集合类)：通过getElementsByTagName等获取的元素集合都是他的一个实例；
  • NodeList(节点集合类)：通过getElementsByName等获取的节点集合都是他的一个实例；
  • HTMLDivElement：（每一个html标签都有一个自己对应的类）-> HTMLElement、Element、Node、EventTarget...
  1. 实例：-> 对象数据类型；
  • 每一个数组都是Array这个类的一个实例；

  • 每一个函数都是Function这个类的一个实例；（无论是实名还是匿名、自执行函数）；

      var ary = new Array();
console.log(typeof ary);//"object" -> 实例ary是一个对象
1. 单例模式（也叫命名空间、nameSpan）
  1. 本质：其实就是一个对象；
  2. 目的：解决命名冲突的问题，模块化方法代码管理；
  3. 使用程度：常用；
  4. 调用方式：
    • 单例模式下，在对象花括号外面用对象名.属性名();调用；

  • 单例模式下，每个属性和属性之间的调用使用this.其他属性名();就可以了；

    • 例如：在person1.fn中调用fn2 -> 就直接在fn1的函数中写this.fn2即可；
    var person1={
     fn1:function () {
         console.log(this);//fn1里面的this是person1 -> 因为是用person1.fn1()调用的，点前面是person1；
         this.fn2(); //  this是person1 -> person1.fn2()
     },
     fn2:function () { }
};
     person1.fn1();//this是person1
     // var f=person1.fn1;   //让fn1中的this变成window
     // f();//this -> window
2. 构造函数模式（类）
  1. 目的：为了创建一个自定义类，并且创建这个类的实例；

  2. instanceof ：专门判断一个实例是否属于某一个类；-> 返回布尔值true/false；

      console.log([] instanceof Array);//true
console.log((function () {}) instanceof Function);//true
console.log(Array instanceof Function);//true   说明Array这个类也是一个函数,是Function类的一个实例；
console.log(Function instanceof Object);//true   说明Function这个类也是Object类的一个实例；

  3. 任何一个引用数据类型都是Object这个类的一个实例；

  4. 定义一个类：function 类名(){} -> 跟定义函数相同；
  • 构造一个类，尽量使用首字母大写；
  • 构造函数中的this就是当前实例 -> 如果想在实例上添加私有属性 -> this.属性名
  • 构造函数会默认返回一个实例this，但是不能在函数中再写一个return一个引用数据类型；
  1. 函数三种角色：
  • 普通函数
  • 构造函数（类）
  • Function这个类的一个实例（对象）
function Person(name,age) {  //首字母大写->构造函数
    //1.系统会自动创建一个对象：this
    //2.添加属性和方法
    this.name=name;  //this是外面接收的实例person1
    this.age=age;
    this.getName=function () {
        alert("我的名字是"+this.name);
    };
    //3.系统默认帮我们返回一个对象this   //构造函数中的this是实例；（因为构造函数中默认返回的this被外面的实例接收了）
}
var person1=new Person("kim",18);
var person2=new Person("lee",19);
person1.getName(); //我的名字是kim
person2.getName(); //我的名字是lee
console.log(person1.getName===person2.getName);//false   说明getName是person1和person2中私有的，开辟了两个空间地址，person1和person2只能调用自己空间中的getName。
